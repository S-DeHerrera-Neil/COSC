# Asking for help via supported CLI inputs:
  - PWSH: Get-Help(Displays help)
  - BASH: man(Same as Get-Help)
  - CMD: cmd /c help(Display cmd commands)

# Cmdlet syntax:
  - (indicates a parameter)

  <> (Indicates Arguments)

  [] (Argument accepts multiple values)
